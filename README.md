## Technology Stack

This project is built using a modern, robust full-stack architecture, combining the power of Laravel with a reactive Vue.js frontend.

### Frontend

- **[Vue.js 3](https://vuejs.org)**: The progressive JavaScript framework.
- **[Tailwind CSS 4](https://tailwindcss.com)**: A utility-first CSS framework for rapid UI development.
- **[PrimeVue 4](https://primevue.org)**: A comprehensive UI component library for Vue, featuring a wide range of accessible components.
- **[Pinia](https://pinia.vuejs.org)**: The intuitive, type-safe store for Vue.js state management.
- **[Chart.js](https://www.chartjs.org)**: Simple yet flexible JavaScript charting for data visualization.
- **[PrimeIcons](https://primevue.org/icons)**: The official icon library for PrimeVue.

### Backend

- **[Laravel 12](https://laravel.com)**: The PHP framework for web artisans, providing a robust backend foundation.
- **[Inertia.js](https://inertiajs.com)**: The modern monolith glue, allowing you to build single-page apps using classic server-side routing and controllers.
- **[Ziggy](https://github.com/tighten/ziggy)**: Use your Laravel routes directly in your JavaScript.

### Development Tools

- **[Vite](https://vitejs.dev)**: Next Generation Frontend Tooling.
- **[Pest](https://pestphp.com)**: An elegant PHP testing framework.
- **[Laravel Pint](https://laravel.com/docs/pint)**: An opinionated PHP code style fixer.

### Development Packages

The following **[Spatie](https://spatie.be/open-source)** packages are planned for integration to enhance functionality:

- **[Laravel Permission](https://spatie.be/docs/laravel-permission)**: For handling user roles and permissions.
- **[Laravel Media Library](https://spatie.be/docs/laravel-medialibrary)**: For associating files with models.
- **[Laravel Activity Log](https://spatie.be/docs/laravel-activitylog)**: For tracking user activity and changes.
- **[Laravel Backup](https://spatie.be/docs/laravel-backup)**: For backing up the application.
